# TempShare 🚀

A simple, database-less temporary file-sharing tool built with Node.js and Express.

## Features
- 📤 **Fast Upload**: Upload any file (up to 50MB).
- 🔑 **Short Access Codes**: Generates a 6-character code for easy sharing.
- ⏱️ **24-Hour Expiry**: Files are automatically deleted after 24 hours.
- 🧹 **Auto-Cleanup**: Background job ensures expired files are removed.
- 🎨 **Modern UI**: Responsive and clean interface.

## Tech Stack
- **Backend**: Node.js, Express, Multer
- **Frontend**: Vanilla JS, CSS3, HTML5

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Access at `http://localhost:3000`
